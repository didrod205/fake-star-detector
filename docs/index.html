fake-star-detector ⭐

Is this GitHub repo's hype real?

Crawls a repo's recent stargazers and flags bought/bot stars — the timing spikes and the empty accounts behind them. Public data only, nothing about you is sent anywhere.

add a GitHub token (optional — raises the rate limit)

Without a token, GitHub allows ~60 checks/hour from your browser. A read-only token raises it to 5,000. It never leaves your browser except to call the GitHub API directly.
